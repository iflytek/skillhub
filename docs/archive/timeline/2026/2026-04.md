# 2026-04

## 摘要

2026 年 4 月显示出仓库已经从快速扩展功能面，转入文档成熟、工作流自动化、兼容性修复，以及围绕真实用户路径进行反复硬化的阶段。当前历史中，这个月共有 `102` 个提交。

## 关键事件

### 2026-04-01 至 2026-04-02

公开文档面在这两天继续扩大：用户手册更新、双语 VitePress 文档站、简化后的 runtime 指引，以及 Kubernetes 部署指南陆续出现。与此同时，e2e 和 PR 校验流程也被加强。

同期，GitHubDaily（4 月 1 日）和 JackCui（4 月 2 日）相继发布微信文章介绍 SkillHub，是最早的一批外部媒体报道。JackCui 的文章从 Claude Code 源码泄露事件切入，深度分析了 Skill 投毒安全风险，将 SkillHub 的审核机制定位为企业级安全管控方案。
