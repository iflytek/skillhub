# 项目起源

## 当前工作性摘要

从现有材料来看，SkillHub 将自己定位为企业级、自托管、面向可复用智能体技能的注册中心。最早一批相对稳定的书面材料，基本都围绕三个互相联动的目标展开：

- 让技能能够在组织内部被发布和发现
- 保留治理能力、审计能力和命名空间边界
- 在合适的地方保持对既有技能打包方式和 CLI 习惯的兼容
