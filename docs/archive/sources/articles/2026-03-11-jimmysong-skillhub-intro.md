# Jimmy Song AI OSS Landscape 收录 SkillHub

## 这是什么

Jimmy Song 在其维护的 AI 开源软件全景图（AI OSS Landscape）项目中，将 SkillHub 作为独立条目收录，归类在 "Dev Tools" 和 "CLI" 类别下。

这是一个结构化的项目介绍页面，包含项目概览、核心特性、使用场景和技术亮点等完整信息。

## 为什么重要

- 这是 SkillHub 首次被纳入第三方维护的 AI 开源生态全景图
- Jimmy Song 是云原生和 AI 领域的知名技术布道者，其全景图项目具有较高的行业影响力
- 该收录为 SkillHub 在企业级 AI agent 技能管理领域的定位提供了外部验证
- 页面提供了结构化的项目信息，便于潜在用户快速了解 SkillHub 的价值主张

## 关键事实

### 项目定位
- 描述为 "open-source, enterprise-grade agent skill registry"
- 强调企业级、自托管、治理能力

### 核心特性摘要
- 自托管部署（Docker/Kubernetes）
- 语义化版本管理
- 团队命名空间与 RBAC
- 评审与治理流程
- CLI 与 REST API
- 可插拔存储后端
- 国际化支持

### 技术栈
- 后端：Spring Boot 3.2.3、Java 21、PostgreSQL 16、Redis 7
- 前端：React 19、TypeScript、Vite、TanStack Router/Query
- 运维：Prometheus + Grafana、Kubernetes

### 使用场景
- 企业内部技能包集中管理
- 为 OpenClaw、AstronClaw、Loomy 等平台提供注册后端
- 多团队协作与分层治理
- 大规模 Kubernetes 环境下的技能仓库
