# 模板

本目录包含项目档案各类条目的模板文件，用于统一归档格式。

## 可用模板

| 模板 | 文件 | 用途 |
|------|------|------|
| 时间线条目 | [timeline-entry.md](./timeline-entry.md) | 记录特定时间段的事件 |
| 里程碑 | [milestone.md](./milestone.md) | 记录项目关键节点 |
| 决策 | [decision.md](./decision.md) | 记录重要决策及其背景 |
| 贡献者 | [contributor.md](./contributor.md) | 记录项目贡献者信息 |
| 沟通摘要 | [communication-summary.md](./communication-summary.md) | 记录会议、讨论等沟通内容 |
| 发文 | [publication.md](./publication.md) | 记录对外发布的内容 |
| 来源 | [source.md](./source.md) | 记录档案信息的来源 |

## 使用方法

1. 复制对应模板到目标目录
2. 按模板中的占位符填写内容
3. 确保填写 frontmatter 中的必填字段（`id`、`type`、`status`）
